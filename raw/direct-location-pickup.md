# Direct Location Pickup Process

## What This Is

Direct Location Pickup is for orders where **items are picked directly from a specific location** and loaded immediately for delivery. This skips the normal warehouse picking process. Usually used for will-call orders or customer pickups.

## When You Use It

- **Customer pickup orders** (will-call)
- **Orders from a single location** that don't need sorting
- Orders marked as "Direct Location Pickup" in the system
- When customer is picking up items themselves

## What Happens Automatically

When a manager creates a Direct Location Pickup ticket:
- System finds all items in the specified location
- Creates a simple picking list
- Assigns it to the pickup location
- Marks it ready for processing

**What you'll see**: Direct Location Pickup Ticket ready in your list.

## How to Process Direct Location Pickups

### Step 1: Find the Ticket

1. Open **Direct Location Pickup Tickets** from your menu
2. Look for tickets with today's date
3. Ticket shows:
   - Customer name
   - Order number(s)
   - Pickup location
   - Shipment method

### Step 2: Review the Order

1. Open the ticket
2. Check customer name - confirm who's picking up
3. Check location - where items are stored
4. Review the items list

### Step 3: Pick and Process Items

**For each item:**

1. Go to the location shown (usually one location for all items)
2. Pick the items
3. **For each line**, enter:
   - **Quantity Picked** - What you actually picked
   - **Quantity Cut** - Any shortfall
   - **Reason Code** if there's a cut

4. Place items in designated staging area or give directly to customer

### Step 4: Record Carts Processed

If items are on carts:
1. Count how many carts were processed
2. Enter **Total Carts Processed** field
3. This helps track cart usage

### Step 5: Complete the Order

1. Verify all items are picked
2. Total Units Picked shows automatically
3. Total Units Cut shows shortfalls (if any)
4. Update **Status** when ready

### Step 6: Customer Pickup (if applicable)

**When customer arrives to pick up:**

1. Verify customer identity
2. Show them the items
3. Let them review quantities
4. Get signature if required
5. Mark order as complete

## Tips for Success

✓ **Check ID** - Verify customer identity for pickups  
✓ **One location** - Usually all items come from the same spot  
✓ **Count carefully** - Customer may be watching  
✓ **Stage clearly** - Keep pickup orders in designated area  
✓ **Update status** - Mark complete when customer has items

## Common Issues

### Item not at the location

**What to do:**
1. Double-check the location code
2. Ask supervisor - item may have been moved
3. Check if there's inventory at other locations
4. Don't make customer wait - get supervisor help quickly

### Customer wants more than ordered

**What to do:**
1. Check if extra inventory is available
2. Get supervisor approval first
3. Don't give items not on the order without approval
4. Supervisor can add items to the order if available

### Customer disputes quantity

**What to do:**
1. Recount together with customer
2. If short, check if items are in another location
3. Get supervisor to help resolve
4. Update cut quantity with reason if short

### No one available at pickup time

**What to do:**
1. Stage items in secure area
2. Note the time customer was expected
3. Inform supervisor
4. Wait for instructions - may need to reschedule

## Where to Find This

**Menu:** Picking → Direct Location Pickup Tickets  
**Page Name:** Direct Location Pickup Ticket  
**Used by:** Warehouse Staff, Customer Service

## What Happens Next

After pickup is complete:
1. System updates inventory
2. Order is marked as fulfilled
3. Customer takes items with them
4. Sales order is ready to be invoiced

For non-pickup direct orders:
1. Items go to loading area
2. Loaded directly on designated truck
3. Delivered to customer
