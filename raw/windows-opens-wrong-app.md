# When I open an attached file with the extension of png or jpeg, it opens as a text file instead of a photo file

## Answer

### 1. **Check File Association Settings**
Ensure that image file types (PNG, JPEG, etc.) are associated with an image viewer/editor by default:
- **On Windows**:
  1. Right-click on a PNG or JPEG file.
  2. Select "Open with" > "Choose another app".
  3. Check the box that says "Always use this app to open.png/.jpeg files".
  4. Select an image viewer (e.g., Photos, Paint, or a third-party image viewer) from the list or browse for one.
- **On macOS**:
  1. Right-click (or Ctrl-click) on a PNG or JPEG file.
  2. Select "Get Info".
  3. Under "Open with", select an appropriate image viewer/editor from the dropdown menu.
  4. Click "Change All..." and confirm to set this application for all files of this type.

### 2. **Reset Default Apps**
Sometimes, resetting default apps can resolve inconsistencies:
- **On Windows**:
  1. Go to Settings > Apps > Default Apps.
  2. Click on "Reset" under "Reset to the Microsoft recommended defaults".
- **On macOS**, the process involves resetting the LaunchServices database, which can be more complex and is generally not recommended unless you're comfortable with Terminal commands. Instead, focus on setting the default app as described above.

### 3. **Use a Different Email Client or Method to Open**
If the issue only occurs when opening attachments from a specific email client:
- Try downloading the attachment and then opening it with your default image viewer.
- Use a different email client to see if the issue is client-specific.

### 4. **Check for Software Updates**
Ensure your operating system and all applications (especially your email client and image viewers) are up to date, as updates often include fixes for file handling issues.

### 5. **Scan for Malware**
Malware can sometimes cause file association problems. Run a full scan with your antivirus software to ensure your system is clean.

### 6. **Reinstall Image Viewers or Email Clients**
If a specific application (viewer or email client) is causing the issue, try uninstalling and then reinstalling it. This can reset the application's settings to default, potentially resolving the issue.

### 7. **System Restore or Seek Support**
- If you've recently installed new software or updates and then noticed the issue, consider using System Restore (on Windows) to revert to a previous point when the issue didn't exist.
- If none of the above steps resolve the problem, it might be time to seek support from the manufacturer of your computer, the email client's support team, or a professional IT service, as there could be a deeper, more specific issue at play.
