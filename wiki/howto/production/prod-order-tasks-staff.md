# Production Order Tasks — Staff Guide

Production order tasks are checklists that ensure quality and consistency in production operations.

## Accessing Your Tasks

### Method 1: From Production Order
1. Open **Released Production Order**
2. Look for **Tasks FactBox** on right side
3. Click **Related → Tasks** for full list

### Method 2: Task Worksheet (All Orders)
1. Search for **Production Order Task Worksheet**
2. Shows ALL tasks across all orders
3. Filter by:
   - **Location Code** (your facility)
   - **Due Date** (today, overdue, etc.)
   - **Prod. Order No.** (specific order)
   - **Completed** (show only incomplete)

## Understanding Task Types

### Headlines (Bold Text)
**Purpose:** Section headers organizing tasks into phases

**Example:** "Preparation Phase", "Quality Checks", "Completion"

**What to do:** Nothing — just read for context (automatically approved)

### Tasks (Indented)
**Purpose:** Actual work items requiring completion

**Example:** "Inspect raw materials", "Clean work area", "Final quality check"

**What to do:** Complete the work, then mark as completed

### Information Lines (Indented)
**Purpose:** Reference information, notes, special instructions

**Example:** "Refer to SOP-123", "Safety note: wear gloves"

**What to do:** Read and follow, but no checkbox to complete

## Completing Tasks

### Step-by-Step

1. **View Available Tasks**
   - Only released tasks shown (approved for you)
   - Released = manager approved

2. **Check Due Date**
   - Focus on overdue tasks first (highest priority)
   - Then work on today's due tasks
   - Plan ahead for upcoming

3. **Complete the Work**
   - Read task description carefully
   - Follow special instructions (check Information lines)
   - Perform required work
   - Verify quality standards

4. **Mark as Completed**
   - Select task line
   - Click **Mark as Completed** action
   - OR check the **Completed** box

5. **Confirmation Messages**

   **If task not yet due:**
   ```
   "This task is not yet due. Do you really want to mark it as completed?"
   ```
   - Click **Yes** only if actually completed early
   - Click **No** to leave incomplete

   **If task due or overdue:**
   ```
   "Do you want to mark this task as completed?"
   ```
   - Click **Yes** to confirm

   **If already completed:**
   ```
   "This Task is already marked as completed. Do you want to undo that?"
   ```
   - Click **Yes** to reopen

6. **Verify Completion**
   - Completed tasks show checkmark
   - System records completion date

### Important Notes

- Cannot complete Headlines (just headers)
- Cannot complete Information lines (reference only)
- Only complete when work is actually done
- Don't mark complete before due unless truly finished early

## Understanding Due Dates

### How Due Dates Work

Automatically calculated from:
- Production order due date
- Routing operation dates
- Production line start/end dates
- Time formulas set by managers

### Due Date Status

| Status | Meaning | Action |
|--------|---------|--------|
| **Future Date** | Not yet due | Can complete early if ready |
| **Today** | Due today | Complete today |
| **Past Date** | Overdue | Complete immediately |

### Common Due Date Patterns

```
Preparation tasks:     2-3 days BEFORE production start
Quality checks:        DURING production (specific routing steps)
Completion tasks:      SAME DAY as production end
Cleanup tasks:         AFTER production complete
```

### If Can't Meet Due Date

1. Notify supervisor immediately
2. Don't mark as complete if it's not done
3. Supervisor may adjust due dates or priorities
4. Follow escalation procedures for critical delays

## Common Scenarios

### Starting New Production Order
1. Review all tasks to understand full workflow
2. Start with first uncompleted task due today or overdue
3. Work through tasks in order (top to bottom)
4. Mark each completed as you finish

### Multiple Orders at Once
1. Filter by **Due Date** to see overdue
2. Complete all overdue tasks first
3. Then filter for today's due tasks
4. Organize by priority or location

### Task Completed Before Due Date
1. Complete the work
2. Click **Mark as Completed**
3. Confirm with "Yes" when warned it's not yet due
4. Okay to work ahead

### Cannot Complete Task on Time
1. DO NOT mark as completed if work isn't done
2. Report to supervisor immediately
3. Document reason
4. Supervisor will reschedule or reassign
5. Complete when able

### Accidentally Marked Wrong Task
1. Select wrong task
2. Click **Mark as Completed** again
3. Click **Yes** to undo
4. Complete correct task

### Task Description Unclear
1. Read Information lines for context
2. Check for document references (SOP-123, etc.)
3. Ask supervisor for clarification
4. Do not guess

### Extra Work Required Not on List
1. Complete work as needed (safety first)
2. Report to supervisor
3. Supervisor can add manual task for documentation
4. Helps improve future templates

## Task Navigation

### Filtering Tasks

| Filter | Purpose |
|--------|---------|
| **Completed = No** | Show only incomplete |
| **Due Date = Today** | Show today's work |
| **Due Date < Today** | Show overdue |
| **Location Code** | Your facility only |

### Sorting Tasks

| Sort By | Best For |
|---------|----------|
| **Due Date** | Prioritizing urgent work |
| **Line No.** | Following task order |
| **Prod. Order No.** | One order at a time |
| **Description** | Finding specific task |

## Best Practices

### ✓ DO:

- **Read entire task description** before starting
- **Check Information lines** for special instructions
- **Complete tasks in order** (usually important)
- **Mark completed immediately** after finishing
- **Report issues** preventing completion
- **Ask questions** if unclear
- **Work ahead** when time permits
- **Keep work area clean** as indicated

### ✗ DON'T:

- **Don't mark complete until work is done** — critical for quality
- **Don't skip tasks** unless supervisor approves
- **Don't ignore due dates** — they're set for a reason
- **Don't guess** if unsure
- **Don't modify tasks** — report to supervisor
- **Don't mark Headlines or Information** — they can't be completed

## Safety and Quality

### Safety Priorities

1. **If task describes unsafe conditions:**
   - STOP work immediately
   - Report to supervisor
   - Do NOT mark as completed
   - Follow safety procedures

2. **If unsure about safety:**
   - Ask supervisor or safety officer
   - Better safe than sorry
   - Safety comes first

3. **If equipment malfunctioning:**
   - Stop immediately
   - Report to supervisor
   - Do not complete task
   - Supervisor addresses equipment

## Daily Efficiency Tips

### Planning Your Day

1. Check overdue tasks (highest priority)
2. Review today's due tasks
3. Look ahead to tomorrow
4. Group similar tasks
5. Update completion status regularly

### Working Through Tasks

1. Follow the sequence (order matters)
2. Don't skip around unless necessary
3. Complete one order at a time when possible
4. Keep task list visible while working
5. Mark completed immediately

### Communication

1. Report problems early
2. Update supervisor on progress
3. Ask questions about anything unclear
4. Suggest improvements

## Troubleshooting

### Can't See Any Tasks
- Check with supervisor if tasks should exist
- Clear any filters (reset view)
- Verify correct production order
- Ask supervisor to release tasks if they exist

### Can't Mark Task Complete
- Check if task is a Headline (can't complete)
- Check if Information line (can't complete)
- Only Tasks can be completed
- Contact supervisor if definitely a Task

### Task Due Date Seems Wrong
- Report to supervisor if impossible
- Supervisor can adjust if needed
- Due dates auto-update if production dates change

### Task List Changed or Disappeared
- Contact supervisor immediately
- Check for new instructions
- New tasks may have been added
